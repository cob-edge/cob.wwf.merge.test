How Does the Simulator and Blockchain Work?

Team Edge have used the technology of the future to devise a system to help you park faster and in the most cost effective manner. We have created a decentralised vehicle parking system, using the secure Ethereum Blockchain Network. Current parking systems are expensive and tedious, and are not designed for the current environment we live in. Edge Parking provides more.

The Ethereum Network

Allows Mining and recording data via a peer to peer decentralised blockchain network

The Simulator

Generating and posting data that mimics actual data from a IoT enviroment

Cloud Database

Recording and savings data

The Network and Key Data Transfers
